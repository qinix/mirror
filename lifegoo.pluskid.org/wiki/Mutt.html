命令行工作

Mutt 可以不启动交互界面直接由命令行参数指定的信息来发送邮件，这个在脚本 里面很有用。因为 Unix 自带的 mail 命令不会帮你处理附件，所以有时候在脚 本里面要发送附件就比较麻烦1，这时可以直接用 mutt 来发送。例如，我的系 统每隔一段时间会自动运行这个脚本，把我的 wiki 备份起来发送到我的专门用 于备份资料的邮箱里面去：

#!/bin/zsh

# my wiki source ,published html along with the
# config files and send it to my gmail

MAILTO="pluskid.archive@gmail.com"
SUBJECT=kid-wiki-$(date +"%G-%m-%d")
ATTACH=${SUBJECT}.tar.bz2


cd /tmp
tar cjf ${ATTACH} -C ${HOME} \
    web/wiki/{src,publish} emacs/init.d/{50muse.el,60muse-latex2png.el,60muse-srctag.el}
mutt -a ${ATTACH} -s $SUBJECT $MAILTO <<EOF
pluskid's wiki backup file in attachement
EOF
